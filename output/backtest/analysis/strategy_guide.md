# 多因子量化选股策略 — 学习指南

> 写给不熟悉量化交易的你。读完这份材料，你会理解自己的策略在做什么、为什么有效、以及如何优化。

---

## 第一章：先理解几个关键概念

### 什么是「年化收益 CAGR」？

假设你投了 10 万，10 年后变成 107 万（总收益 970%）。CAGR 回答的问题是：**如果每年稳定增长，需要多少增速才能达到这个结果？**

答案是 26.5%。意思是「平均下来每年赚 26.5%」，但实际上有的年赚 50%，有的年亏 10%。

作为参考：
- 巴菲特长期 CAGR 约 20%
- 标普 500（SPY）长期 CAGR 约 10-12%
- 你的策略回测 CAGR 26.5%，优化后 27.8%

### 什么是「最大回撤 Max Drawdown」？

**你最惨的时候从高点跌了多少。**

举个例子：
```
1月：100万
3月：涨到 150万  ← 高点
6月：跌到  98万  ← 低点
9月：回到 160万
```

最大回撤 = (150 - 98) / 150 = **34.7%**

这个数字很重要，因为它代表你需要承受的**心理压力**。回撤 35% 意味着你可能看着 150 万变成不到 100 万，持续好几个月。很多人会在这个时候恐慌卖出，错过后面的反弹。

你的策略：
- 当前 Max DD：36.6%（最惨跌了三分之一多）
- 优化后 Max DD：31.8%（好了一些，但仍然需要心理准备）

### 什么是「Sharpe Ratio（夏普比率）」？

**每承受一单位风险，能赚多少超额回报。**

公式很简单：Sharpe = (策略收益 - 无风险收益) / 波动率

- Sharpe < 0.5：不太好
- Sharpe 0.5-1.0：还行
- Sharpe 1.0-1.5：很好
- Sharpe > 1.5：极好（很少见）

你的策略 Sharpe 1.04 → 优化后 1.13，属于「很好」的水平。

### 什么是「Alpha」？

**你比市场多赚的部分。**

如果 SPY 涨了 15%，你涨了 26%，那中间的差值（扣除风险调整后）就是 Alpha。Alpha > 0 说明你的选股能力在创造价值，而不是纯粹靠承担更多风险。

你的策略 Alpha 约 10.8%，意思是扣除市场涨幅后，你的选股**每年额外赚了约 10.8%**。

## 第二章：你现在的策略在做什么

### 一句话版本

> 每个月看一遍你的股票池，用「便不便宜 + 涨势好不好 + 大环境如何」打分，买最好的几只，跌太多就止损。

### 展开说

你的策略分五步，每个月重复一次：

**第一步：确定股票池**

你不是从全市场几千只股票里选，而是只看你之前做过 DCF（现金流折现）分析的公司。目前有 17 只，包括 AAPL、NVDA、TSLA、PLTR 等。

这一步就像你先做了功课，列了一个「值得关注的好公司名单」，然后每月从里面挑。

**第二步：给每只股票打分**

策略用 6 个「因子」打分，分成三大类：

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   价值类（便不便宜？）                            │
│   ├─ F1: DCF 折扣 — 现价比你算的公允价值低多少     │
│   └─ F2: 相对估值 — 现价在过去一年是高还是低        │
│                                                 │
│   动量类（涨势好不好？）                           │
│   ├─ F3: 趋势确认 — 股价在 200 日均线上方吗         │
│   └─ F4: 动量 — 过去 6 个月涨了多少               │
│                                                 │
│   周期类（大环境如何？）                           │
│   ├─ F5: 宏观环境 — VIX、利率曲线、SPY 趋势        │
│   └─ F6: 行业周期 — 收入增速是否在加速             │
│                                                 │
└─────────────────────────────────────────────────┘
```

最终合成一个 0-1 的综合分数。分数越高，越值得买。

**第三步：根据大环境决定买多少**

策略会判断市场处于哪种状态：

| 状态 | 判断依据 | 投入比例 | 选多少只 |
|------|----------|----------|----------|
| 进攻 | SPY 在均线上、VIX 低、利率正常 | 100% | 前 40% |
| 中性 | 信号不一致 | 65% | 前 25% |
| 防守 | SPY 跌破均线、VIX 高 | 30% | 前 10% |

这就像开车：晴天踩油门，雨天减速，暴风雪就靠边停。

**第四步：分配仓位**

不是平均分配，而是用「风险平价」：波动大的股票少买，波动小的多买。每只股票最多占 20%。

这样做的好处是不会因为一只暴涨暴跌的股票亏太多。

**第五步：什么时候卖**

触发以下任一条件就卖：

| 条件 | 说明 | 举例 |
|------|------|------|
| 止损 | 从买入价跌了 15% | 100 买的，跌到 85 就卖 |
| 到价 | 涨到你算的公允价值 | DCF 算出值 150，涨到 150 就卖 |
| 排名掉了 | 综合分数不在前几名了 | 别的股票更好，换掉 |
| 趋势破了 | 跌破 200 日均线 | 长期趋势可能转空 |

## 第三章：优化后的策略有什么不同

Phase 3 做的事情是：测试了不同参数组合，找到最优配置。

### 核心变化：因子权重

| | 价值类权重 | 动量类权重 | 周期类权重 |
|---|---|---|---|
| **原来** | 40-50%（随环境变） | 20-35%（随环境变） | 25-30%（随环境变） |
| **优化后** | **70%**（固定） | **30%**（固定） | **0%**（去掉） |

**为什么去掉周期因子？**

通过 66 组权重组合的回测，发现：
- 行业周期因子（F6）对选股的帮助很小
- 加入周期因子反而引入噪声，降低了 Sharpe
- Top 10 最优组合中，周期权重几乎都是 0

直觉上也说得通：你的股票池只有 17 只，行业分散，用一个粗略的周期指标区分不了太多。

**为什么 70/30 而不是 50/50？**

| 组合 | Sharpe | Max DD |
|------|--------|--------|
| 50% / 50% | 1.04 | 31.8% |
| 60% / 40% | 1.12 | 31.8% |
| **70% / 30%** | **1.13** | **31.8%** |
| 80% / 20% | 1.10 | 32.1% |
| 90% / 10% | 1.05 | 31.4% |

70/30 是甜蜜点。价值因子太少选不到便宜货，太多又会错过有涨势的股票。

### 其他参数没变

止损 15%、仓位上限 20%、动量看 6 个月、趋势看 200 日均线 — 这些经过敏感性分析，发现当前默认值已经接近最优，不需要调。

## 第四章：新旧策略对比

### 回测数据对比（2016-2026，10 年）

| 指标 | SPY 买了不动 | 原策略 | 优化后策略 | 说明 |
|------|-------------|--------|-----------|------|
| **总收益** | 206% | 970% | ~1000% | 10万→30.6万 vs 10万→107万 |
| **年化 CAGR** | 14.9% | 26.5% | 27.8% | 每年多赚 1.3% |
| **Sharpe** | ~0.7 | 1.04 | **1.13** | 风险调整后收益更好 |
| **最大回撤** | ~33% | 36.6% | **31.8%** | 最惨时少跌近 5% |
| **Alpha** | 0% | 10.8% | 11.6% | 跑赢大盘的幅度 |

### 用人话说

- **收益差不多**：CAGR 从 26.5% 到 27.8%，差 1.3%，10 年累积下来大概多赚 15%
- **风险明显下降**：最大回撤从 36.6% 降到 31.8%，意味着你最惨的时候少亏约 5 万（按 100 万本金算）
- **综合性价比更高**：Sharpe 从 1.04 到 1.13，每承受一单位风险多赚 8.6%

优化的意义不是「赚更多」，而是「同样的收益，承受更少的痛苦」。

## 第五章：如何执行

### 如果你想用优化后的策略跑回测

```bash
# 方法 1：直接跑分析看结果
python -m quant.run_analysis --analysis single_factor

# 方法 2：在代码中指定
from quant.config import BacktestConfig

config = BacktestConfig(
    factor_weights={"value": 0.70, "momentum": 0.30, "cycle": 0.0},
    stop_loss_pct=0.15,          # 止损 15%
    max_position_weight=0.20,    # 单只上限 20%
    momentum_lookback=126,       # 动量看 6 个月
    sma_window=200,              # 趋势看 200 日均线
    label="optimal_v70_m30",
)
```

### 如果你想在实盘中手动执行

每月第一个交易日，花 30 分钟做以下事情：

```
1. 更新你的 DCF 分析（每季度财报后更新一次即可）

2. 检查大环境：
   □ SPY 在 200 日均线上方吗？
   □ VIX 低于 25 吗？
   □ 10年-2年利率差 > 0 吗？
   → 2-3 个"是"= 进攻，1 个"是"= 中性，0 个"是"= 防守

3. 给每只股票打分：
   - 70% 权重看「便不便宜」（DCF 折扣 + 历史价位）
   - 30% 权重看「涨势好不好」（是否在均线上 + 6 个月涨幅）

4. 买入/调仓：
   - 进攻模式：买分数最高的 6-7 只（17只的40%）
   - 中性模式：买最高的 4 只
   - 防守模式：只买最高的 1-2 只，其余持现金

5. 检查持仓：
   - 跌 15% 的 → 止损卖出
   - 涨到 DCF 公允价值的 → 获利了结
   - 排名掉出前列的 → 换股
```

## 第六章：注意事项

### 回测 ≠ 未来

回测用的是 2016-2026 的历史数据。这 10 年包含了美股大牛市（2016-2019）、疫情暴跌和反弹（2020）、加息周期（2022-2023）。策略在这些环境下都表现不错，但不保证未来一样。

### 你的 Alpha 主要来自哪里

坦白说，大部分 Alpha 来自**你的股票池本身**。NVDA、PLTR、TSLA、AVGO 这些公司过去 10 年表现远超大盘。策略的作用是：
1. 在它们便宜的时候多买（价值因子）
2. 在它们下跌时少亏（止损 + 防守模式）
3. 在它们不行时及时换掉（排名轮换）

如果你的股票池里全是平庸的公司，再好的因子权重也不会创造奇迹。

### 回撤是真实的痛苦

31.8% 的 Max DD 看起来是个数字，但想象一下：

> 你年初有 150 万。到了 6 月，账户只剩 102 万。每天打开 app 都在亏钱，新闻都在说经济要崩了。你的策略说「继续持有」。你能忍住不卖吗？

如果不能，可以考虑：
- 降低仓位上限（从 20% 降到 15%）
- 收紧止损（从 15% 降到 10%）
- 代价是收益会略微下降

### 交易成本

回测已经计入了每笔交易 0.15% 的成本（佣金 + 滑点）。月度调仓频率不算高，交易成本影响有限。
