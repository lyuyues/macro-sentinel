# DIY vs Hire — 装修任务拆分

## WA 州法律前提

- **电工活**：房主可以自己做自住房的电气工程，但必须买 permit + 叫 inspection
- **水管活**：房主可以自己做自住房的管道工程，同样需要 permit + inspection
- **结构性改动**：需要 permit，可能需要工程师签字
- **石棉/铅漆处理**：必须持证专业公司，不能 DIY
- **油漆/地板/拆除/装柜子**：无需 permit，自由 DIY

## SCOPE A: 壁炉拆除

| 任务 | DIY? | 难度 | 说明 |
|------|------|------|------|
| 拆 insert/firebox | **DIY** | ★★☆ | 重体力活，需要撬棍、锤子、护目镜 |
| 拆 hearth 石材/砖 | **DIY** | ★★☆ | 用电锤/手锤，注意粉尘 |
| 拆烟囱内部 chase | ⚠️ 看情况 | ★★★ | 如果 load-bearing 必须请人 |
| 拆屋顶烟囱 + 补屋顶 | **请人** | — | 涉及屋顶防水，做错代价大 |
| 断开 gas line | **请人** | — | 必须持证 plumber |
| 断开电线 | **DIY** (permit) | ★★☆ | 关总闸，cap wires，叫 inspection |
| 石棉检测 | **请人** | — | $200-500 检测费，法律要求 |
| 补墙（framing + drywall） | **DIY** | ★★★ | 需要学 framing 和 drywall finishing |
| 补地板 | **DIY** | ★★☆ | 取决于材料是否能配上 |
| 油漆 | **DIY** | ★☆☆ | 最容易的部分 |

### DIY 节省估计
- 全部请人：$5,000–$12,000
- 自己拆除 + 补墙 + 油漆，只请人做屋顶/gas/结构：**$2,000–$4,000**
- **省 $3,000–$8,000**

### 需要买/租的工具
- 电锤 (rotary hammer) — 租 ~$60/天
- 撬棍、锤子、护目镜、N95 口罩、手套
- Drywall 工具套装（tape, mud, knife）~$50
- 垃圾清运 dumpster 租赁 ~$350/周

| 装灯/镜子/配件 | **DIY** | ★☆☆ | 最简单 |
| 油漆 | **DIY** | ★☆☆ | |

### DIY 节省估计
- 全部请人（中档）：$30,000–$50,000
- 自己拆除 + 贴砖 + 装 fixture，只请人做管道粗管 + 电气粗管：**$12,000–$20,000**（主要是材料费）
- **省 $18,000–$30,000**

### 时间投入
- DIY 主浴室：预计 **4–8 个周末**（如果每周末 8-10 小时）
- 贴砖是最大时间消耗

## SCOPE E: 天花板装灯

| 任务 | DIY? | 难度 | 说明 |
|------|------|------|------|
| 评估天花板结构 | **DIY** | ★☆☆ | 上阁楼看看 |
| 买 permit | **DIY** | ★☆☆ | 在线申请 homeowner electrical permit |
| 从 panel 拉线到阁楼 | ⚠️ **DIY 或请人** | ★★★★ | 最难的部分，需要了解 panel 空间 |
| 阁楼里走 Romex 到各灯位 | **DIY** | ★★★ | 爬阁楼、钻孔、固定线 |
| 木条上开洞装 recessed can | **DIY** | ★★★ | 用 hole saw，小心别裂木头 |
| 墙上开洞装 switch box | **DIY** | ★★★ | 用 drywall saw |
| 接线（switch + fixture） | **DIY+inspection** | ★★★ | 学好接线规范，用 wire nut |
| 装 dimmer switch | **DIY** | ★★☆ | 接 3 根线 |
| 叫 inspection | **DIY** | ★☆☆ | 网上约 inspector |

### DIY 节省估计
- 全请电工：$8,000–$18,000
- 自己走线+装灯+装 switch，只请电工做 panel 接线 + final inspection：**$2,000–$5,000**（材料+permit+电工 panel 工时）
- **省 $6,000–$13,000**

### 提醒
- 如果有阁楼空间，难度降低一半
- 没有阁楼的话，走线要拆天花板再封，建议请人
- **先从一个房间试手**，学会了再做其他房间

## SCOPE F: 外立面

| 任务 | DIY? | 难度 | 说明 |
|------|------|------|------|
| 拆旧 siding | **DIY** | ★★☆ | 体力活，需要梯子/脚手架 |
| 检查/修 sheathing 腐烂 | **DIY** | ★★★ | 局部换木头 |
| 装 house wrap (Tyvek) | **DIY** | ★★☆ | staple gun |
| 装 rainscreen furring strips | **DIY** | ★★★ | 需要整齐钉 |
| 装 Hardie siding | ⚠️ **可 DIY 但很辛苦** | ★★★★ | 材料重、需要切割、二楼需要脚手架 |
| Window/door flashing | ⚠️ **关键节点** | ★★★★ | 做错会漏水，建议请人或学透再做 |
| 装 trim (角/窗框/fascia) | **DIY** | ★★★ | |
| 油漆（如果非 ColorPlus） | **DIY** | ★★☆ | |
| 脚手架搭建 | **租** | — | 租脚手架 ~$200-400/月 |

### DIY 建议
外立面是 **最不建议全 DIY** 的 scope：
- 二楼高空作业有安全风险
- Flashing 做错 → 漏水 → 墙体腐烂 → 几万块修
- Hardie board 很重（一片 ~30 磅），切割产生硅尘（需要呼吸防护）
- 保修需要按厂家规范安装

**建议**：拆旧 siding 自己做（省 $3,000-5,000），新 siding 安装请人。

## 必须请人的红线（不要省）

1. **Gas line** — 任何涉及天然气的必须持证
2. **屋顶修补**（壁炉拆除后）— 漏水代价太大
3. **石棉处理** — 法律要求 + 健康风险
4. **台面安装** — 几百磅的石板，需要专业搬运切割
5. **Siding flashing** — 做错会导致墙体腐烂
6. **Panel 电箱接线** — 可以自己走线，但主箱接线让电工做

## DIY 优先级（从最容易开始练手）

```
Level 1 (先做这些，建立信心):
  → 油漆（卧室、Office）
  → 换马桶
  → 换水龙头 / vanity
  → 换门把手 / 面板
  → 铺 LVP 地板

Level 2 (有了经验再做):
  → 拆除（壁炉、旧柜子、旧瓷砖）
  → 装厨房柜子
  → 贴 backsplash
  → 装灯 / 走线（单个房间先试）

Level 3 (需要认真学习):
  → 浴室贴砖 + 防水
  → 浴室地暖安装
  → 全屋天花板走线
  → 厨房电路布线

请人做:
  → Gas line / 台面 / 屋顶 / siding 安装 / 石棉
```
