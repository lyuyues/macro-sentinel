## SCOPE A: Fireplace & Chimney Removal

### A1. Pre-Work / Assessment
- [ ] Determine fireplace type: wood-burning / gas / electric

- [ ] **Asbestos testing** of chimney flue lining, insulation, and surrounding materials (required before demolition)
- [ ] Structural assessment: is chimney load-bearing? Does it support any roof structure?

### A2. Demolition

- [ ] Remove fireplace insert / firebox
- [ ] Remove hearth and surround (tile / stone / brick)
- [ ] Remove chimney chase from interior (floor to ceiling)
- [ ] **Option 1 (preferred):** Remove chimney stack above roofline
- [ ] **Option 2 (budget):** Cap chimney at roofline only, leave stack
- [ ] Remove all debris, haul away

### A3. Structural Repair
- [ ] Install structural header/support if load-bearing wall is affected
- [ ] Reframe wall opening where chimney was removed
- [ ] Patch subfloor where hearth was removed
- [ ] Repair/replace roof sheathing and roofing at chimney penetration
- [ ] Install new flashing and ensure waterproof seal at roof patch
- [ ] Insulate any newly exposed exterior wall cavity

### A4. Finishing
- [ ] Drywall new wall surface, tape, mud, texture to match existing
- [ ] Match existing flooring in fireplace footprint area
- [ ] Paint repaired wall area
- [ ] Patch and paint ceiling if affected
- [ ] Final inspection and permit closeout

### A5. Exclusions (unless noted)
- Asbestos abatement (separate licensed contractor if needed)

## SCOPE E: Interior Ceiling Lighting — Wood Plank Ceilings

**Current condition:** Wood plank/tongue-and-groove ceiling throughout, **no existing overhead lights.** Rooms currently rely on floor/table lamps and wall outlets only.

*This is a whole-house scope that should be coordinated across all other scopes.*

### E1. Assessment
- [ ] Inspect ceiling construction: determine what is above the wood planks
  - Attic space above? → easiest, can run wiring through attic
  - Drywall/plaster above planks? → moderate difficulty
  - Direct to roof joists? → most complex
- [ ] Determine plank thickness and material (solid wood vs T&G vs beadboard)
- [ ] Check existing electrical: is there any wiring already run to ceiling locations?
- [ ] Evaluate panel capacity: may need subpanel or circuit additions for added lighting load

### E2. Lighting Approach Options (contractor to recommend based on assessment)

**Option 1: Recessed lights through wood planks (preferred if attic access)**
- Cut holes in wood planks for recessed LED cans
- Run Romex through attic to each fixture location
- Use IC-rated, airtight recessed housings
- Requires careful cutting to avoid splitting wood
- Pro: clean look, flush with ceiling
- Con: visible holes if planks are a design feature you want to preserve

**Option 2: Surface-mount fixtures on wood planks**
- Mount flush-mount or semi-flush LED fixtures directly to planks
- Run wiring through attic or behind planks via junction boxes
- Pro: no holes in planks, easier install
- Con: fixtures are visible, may look less modern

**Option 3: Track or cable lighting mounted to beams/planks**
- Mount track rail along beam lines
- Adjustable heads for directional light
- Pro: flexible, can highlight areas, minimal ceiling penetration
- Con: more visible, industrial aesthetic

**Option 4: Decorative pendant lights**
- Hang pendants from ceiling at key locations (dining, kitchen island)
- Can combine with recessed elsewhere
- Pro: design statement
- Con: limited to specific locations
